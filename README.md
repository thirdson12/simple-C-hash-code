# simple-C-hash-code

Simple hash func 


Ci=bi1 (+) bi2 (+)…..(+) bim

*ci is İth bit of hash code 1<= I <= n
*M is number of n-bit blocks in input
*Bij  is jth bit in jth block
*Probalitiy data error result in unchanged hash value: 2^n
